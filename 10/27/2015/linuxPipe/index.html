Grep

文本搜索 支持正则表达式
grep [option][fileName]
常用option
-i ignore 忽略大小写
-l list 输出包含匹配字符的文件名
-n number 显示匹配内容的行号
-s —no-messages 忽略错误
-r recursive 递归查找
-v inver-match 排除匹配的行
-e regex
//正则就不讲了
example:
cat test.sh | grep -n ‘echo’
outPut:
1: echo “Hello”;

cut

用来截取字符串
cut[option1][option2][fileName]
option1
-d delimiter 分隔符(split)
-f fields 字段列举
-c characters 字符为单位
-b bytes 字节为单位
//其他字符的原因会存在编码大小的问题,英文环境-b === -c

option2(截取范围)
N (N)-第N项
N- N-末尾
N-M (N,M)
-M (1,M)
