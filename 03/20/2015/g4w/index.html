    1. HeroKu
    2. OpenShift
    3. Koding
    4. etc
      原理都一样: remote机器开的ssh可以为本地一个隧道,进行端口转发即可.
      条件: 开放的SSH
      Code:

      ssh -D 7001 username@remote-host

-D 表示动态绑定 7001为本地端口,如果上传了id_rsa.pub,就没有密码登陆了
