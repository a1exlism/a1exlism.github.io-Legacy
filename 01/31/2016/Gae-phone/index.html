借助GAE实现手机爬梯

前提

1.拥有国内VPS或者学生认证资格(可以去腾讯或者阿里租国内服务器，比qq会员还省钱= =)
2.熟悉GoAgent和一丁点的linux基础

正文

下载并启用

在服务器上下载xx-net,找个合适的地方进行git clone https://github.com/XX-net/XX-Net.git
之后root权限启用xx-netcd XX-NET & sudo ./start.sh

xx-net配置

开发大大真的做到了贴心，已经集成了远程管理和后台自启功能，不过服务器系统默认不安装图形界面。
可以照着网上教程安装一遍X界面，然后利用浏览器进行各项配置。

1.初始配置可以使用config.yaml文件进行设置。

文件目录:*/xx-net/data/launcher/config.yaml

modules:
  gae_proxy: {auto_start: 1, current_version: ''}
  launcher: {allow_remote_connect: 0, auto_start: 1, control_port: 8085, current_version: '',
    popup_webui: 0, proxy: gae, show_systray: 1}
  php_proxy: {control_port: 8083}
update: {check_update: stable, last_path: /home/a1exlism/tools/xx-net/launcher, uuid: 30988f10-9cf8-4c76-ba90-cd62d56ff304}

auto_start: 1 自启 allow_remote_connect:1 允许远程控制 control_port等其他的就是字面意思了

2.还有个手工配置

vim ./XX-NET/data/gae_proxy/manual.ini #创建一个手工设置文件
#添加:
[listen]
ip = 0.0.0.0 #保证外网可以访问
port = 8087 #按照选择自行更改端口
visible = 1
debuginfo = 0

原理:编程随想

3.自启动设置

如果webUI无法设置成自启动的话,可以尝试这个

sudo vi /etc/rc.local
#在exit 0之前添加下面这句
/home/a1exlism/tools/XX-NET/start.sh

关闭终端之前保持start.py的运行

(/home/a1exlism/tools/XX-NET/start.sh &)
#类似于js闭包概念 可以无视当前命令=, =

版本:android4.4(已Root) 关键问题:证书问题 android5.1听朋友说好像问题挺大,等过短时间手机换了再去考虑

远程copy证书

#利用ssh中的scp(secure copy)命令进行远程传输
scp root@Your IP:/xx-net/data/gae_proxy/CA.crt  /home/a1exlism/Downloads
#ssh端口如果更改则添加-P参数 -r递归

证书转换

xx-net的ctr证书和android系统证书还是有较大区别的.为了不设置锁屏,而且更加的自由,将xx-net生成的证书进行转换.
1. 获取证书hash

#通过firefox将crt转换成pem格式
openssl x509 -inform PEM -subject_hash_old -in CA.pem | head -1

2.转换crt
cat CA.crt > bbe23816.0   #bbe23816为刚才产生的hash值
openssl x509 -inform PEM -text -in PortSwiggerCA.crt -out /dev/null >> bbe23816.0

证书安装

```bash
adb push bbe23816.0 /sdcard/
adb shell #接入android shell
shell@hwp6-c00:/ $ su
root@hwp6-c00:/ # mount -o remount,rw /system #以读写形式挂载/system目录
root@vbox86p: cp /sdcard/9a5ba575.0 /system/etc/security/cacerts/
root@vbox86p: chmod 644 /system/etc/security/cacerts/9a5ba575.0
